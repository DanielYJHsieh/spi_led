# Arduino UNO 與 aRGB LED 接線圖

## 基本接線 (最簡單的連接方式)

```
Arduino UNO                    aRGB LED (如 WS2812B)
┌─────────────┐               ┌─────────────┐
│             │               │             │
│      Pin 11 ├───────────────┤ DIN         │
│     (MOSI)  │               │             │
│             │               │             │
│         5V  ├───────────────┤ VCC (5V)    │
│             │               │             │
│        GND  ├───────────────┤ GND         │
│             │               │             │
│    Pin 13   │ (未使用)       │ DOUT ───────┤ 
│     (SCK)   │               │             │ (連接下一顆LED)
└─────────────┘               └─────────────┘
```


## 建議的改良接線 (增加穩定性)

```
Arduino UNO                           aRGB LED Strip
┌─────────────┐                      ┌─────────────────┐
│             │     [33Ω]            │                 │
│      Pin 11 ├─────/\/\/\───────────┤ DIN             │
│     (MOSI)  │   保護電阻            │                 │
│             │                      │                 │
│         5V  ├──────┬───────────────┤ VCC (5V)        │
│             │      │ [100µF]       │                 │
│        GND  ├──────┼───||──────────┤ GND             │
│             │      │ [0.1µF]       │                 │
│             │      └───||──────────┤                 │
└─────────────┘        去耦電容        └─────────────────┘

說明:
- 33Ω 電阻: 保護數據線，減少信號反射
- 100µF 電解電容: 濾波大電流變化
- 0.1µF 陶瓷電容: 濾除高頻雜訊
```

## 電源考量接線 (大功率LED條)

```
Arduino UNO              外部電源           aRGB LED Strip
┌─────────────┐         ┌─────────┐        ┌─────────────────┐
│             │         │  5V     │        │                 │
│      Pin 11 ├─────────┼─────────┼────────┤ DIN             │
│     (MOSI)  │  [33Ω]  │ 3-5A    │        │                 │
│             │         │ 電源供應器│        │                 │
│        GND  ├─────────┤   GND   ├────────┤ VCC (5V)        │
│             │         │         │   │    │                 │
│     Vin/5V  │ (可選)  │   5V    │   │    │ GND             │
│             │         └─────────┘   │    │                 │
└─────────────┘                       │    └─────────────────┘
                                      │
                                  ┌───┴───┐
                                  │ 大容量 │
                                  │ 電容   │
                                  │1000µF+ │
                                  └───────┘

注意事項:
- 每30顆LED建議使用外部電源
- Arduino的5V輸出最大電流約500mA
- 每顆LED全亮時約消耗60mA
- 大容量電容防止電壓驟降
```

## 引腳說明

### Arduino UNO SPI 引腳
| 引腳 | 功能 | 連接 | 說明 |
|------|------|------|------|
| Pin 11 | MOSI | LED DIN | 主要數據輸出 |
| Pin 12 | MISO | 未使用 | 單向通訊協議 |
| Pin 13 | SCK | 未使用 | aRGB LED不需要時鐘線 |
| Pin 10 | SS | 未使用 | 非必要 |

### aRGB LED 引腳
| 引腳 | 功能 | 連接 | 說明 |
|------|------|------|------|
| DIN | 數據輸入 | Arduino Pin 11 | 接收控制數據 |
| DOUT | 數據輸出 | 下一顆LED DIN | 串聯下一顆 |
| VCC | 電源正極 | 5V | 5V供電 |
| GND | 電源負極 | GND | 接地 |

## 連接檢查清單

- [ ] 數據線連接正確 (Pin 11 → DIN)
- [ ] 電源連接穩固 (5V → VCC, GND → GND)
- [ ] LED方向正確 (DIN → DOUT 方向)
- [ ] 接線牢固無鬆動
- [ ] 電源供應充足
- [ ] 建議添加保護電阻和電容

## 常見連接錯誤

❌ **錯誤**: 將數據線連接到 Pin 13 (SCK)
✅ **正確**: 將數據線連接到 Pin 11 (MOSI)

❌ **錯誤**: LED方向接反 (DOUT → DIN)
✅ **正確**: LED方向正確 (DIN → DOUT)

❌ **錯誤**: 直接用Arduino 5V為大量LED供電
✅ **正確**: 使用外部電源供應器

❌ **錯誤**: 忽略電源濾波
✅ **正確**: 添加適當的濾波電容
